This Amazon Clone is a front-end static website created using HTML and CSS, inspired by the design and layout of Amazon. The main goal of this project is to practice and demonstrate core web development skills, especially in structuring web pages and styling them effectively.

The website replicates key visual elements of Amazon, including a navigation bar, search bar, product sections, promotional banners, and footer. It is designed to be responsive and visually consistent across different screen sizes.

🚀 Features:
Responsive homepage layout
Navigation bar with logo and search box
Product sections and categories
Banner images and promotional sections
Clean and modern UI design
🛠️ Technologies Used:
HTML5
CSS3

This project serves as a strong foundation for moving into advanced topics like JavaScript, React, and full-stack e-commerce development.
